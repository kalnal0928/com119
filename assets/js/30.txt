AND 연산 기본 원리
AND 연산은 두 비트가 모두 1일 때만 1이 됩니다.

1 AND 1 = 1
1 AND 0 = 0
0 AND 1 = 0
0 AND 0 = 0

단계별 계산
1단계: 16진수를 4비트씩 나누어 생각하기
x10 = 0xFFFF_1234
     F    F    F    F    1    2    3    4
   1111 1111 1111 1111 0001 0010 0011 0100

x11 = 0x0000_FF00  
     0    0    0    0    F    F    0    0
   0000 0000 0000 0000 1111 1111 0000 0000
2단계: 각 자리별로 AND 연산
F AND 0 = 0    →  1111 AND 0000 = 0000 = 0
F AND 0 = 0    →  1111 AND 0000 = 0000 = 0  
F AND 0 = 0    →  1111 AND 0000 = 0000 = 0
F AND 0 = 0    →  1111 AND 0000 = 0000 = 0
1 AND F = 1    →  0001 AND 1111 = 0001 = 1
2 AND F = 2    →  0010 AND 1111 = 0010 = 2
3 AND 0 = 0    →  0011 AND 0000 = 0000 = 0
4 AND 0 = 0    →  0100 AND 0000 = 0000 = 0
3단계: 결과 조합
0 0 0 0 1 2 0 0 = 0x0000_1200
답: x9 = 0x0000_1200
핵심은 x11에서 0인 부분(0x0000과 0x00)이 있으면 AND 결과도 무조건 0이 된다
