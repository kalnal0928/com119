XOR 연산을 이용하여 레지스터를 0으로 초기화하려면 같은 값끼리 XOR 연산을 해야 합니다.
핵심 조건
동일한 두 값을 XOR 연산하면 결과는 항상 0이 됩니다.
A ⊕ A = 0
실제 적용 예시
어셈블리어에서 레지스터를 0으로 초기화할 때:
assemblyXOR EAX, EAX    ; EAX 레지스터를 자기 자신과 XOR → EAX = 0
XOR RBX, RBX    ; RBX 레지스터를 자기 자신과 XOR → RBX = 0
XOR의 특성
이것이 가능한 이유는 XOR 연산의 다음 특성 때문입니다:

0 ⊕ 0 = 0
1 ⊕ 1 = 0

따라서 어떤 비트 패턴이든 자기 자신과 XOR하면 모든 비트가 0이 됩니다.
장점
이 방법이 널리 사용되는 이유:

효율성: MOV 명령어보다 빠름
코드 크기: 더 적은 바이트 사용
범용성: 레지스터의 현재 값에 관계없이 항상 0으로 만듦
