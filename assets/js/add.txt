주관식 43
문제
다음은 4개의 논리 주소를 통해 페이지 기반 가상 메모리 시스템에서 실제 물리 주소 및 데이터를 찾는 과정이다. 다음의 각 질문에 답하시오. • 10진수 논리주소 0, 6, 10, 13에 대한 페이지 번 호 p와 변위 d를 순서대로 2진수로 각각 나타내 시오 • 위에서 구한 각 논리 주소에 맵핑되는 물리 주소 의 프레임 번호 f와 변위 d를 2진수로 각각 답하 시오 • 위에서 구한 4개의 물리 주소에 의하여 물리 메 모리에 access된 4개의 문자 데이터를 차례대로 나열하시오

정답
1단계: 논리 주소 분석 (페이지 번호 p와 변위 d)

논리 주소 0 (0000₂): p = 00₂, d = 00₂
논리 주소 6 (0110₂): p = 01₂, d = 10₂
논리 주소 10 (1010₂): p = 10₂, d = 10₂
논리 주소 13 (1101₂): p = 11₂, d = 01₂

2단계: 페이지 테이블을 통한 프레임 번호 찾기
페이지 테이블에서:

페이지 0 → 프레임 3
페이지 1 → 프레임 5
페이지 2 → 프레임 1
페이지 3 → 프레임 6

물리 주소의 프레임 번호 f와 변위 d (2진수):

논리 주소 0: f = 011₂ (프레임 3), d = 00₂
논리 주소 6: f = 101₂ (프레임 5), d = 10₂
논리 주소 10: f = 001₂ (프레임 1), d = 10₂
논리 주소 13: f = 110₂ (프레임 6), d = 01₂

3단계: 물리 메모리에서 데이터 찾기
물리 주소 계산 (프레임 번호 × 4 + 변위):

논리 주소 0: 물리 주소 = 3×4 + 0 = 12 → U
논리 주소 6: 물리 주소 = 5×4 + 2 = 22 → AA
논리 주소 10: 물리 주소 = 1×4 + 2 = 6 → S
논리 주소 13: 물리 주소 = 6×4 + 1 = 25 → N
